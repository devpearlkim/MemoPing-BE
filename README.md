## API Documentation


### **Endpoints**

- `POST /memos`: Create a new memo.
- `GET /memos`: Retrieve all memos.
- `GET /memos?search=keyword`: Search for memos by keyword.
- `GET /memos?tags=tag1,tag2`: Filter memos by hashtags.
- `DELETE /memos/:id`: Delete a memo by its ID.
